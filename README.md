## Como executar (XAMPP)

Como a maioria dos exemplos é web, a forma mais comum é usar **XAMPP**.

### 1) Hospede o projeto na pasta do servidor
Este repositório está em:

- `c:/xampp/htdocs/Jovem-Aprendiz-Senac/`

Assim, você acessa os arquivos via:

- `http://localhost/Jovem-Aprendiz-Senac/`

### 2) Rodar páginas HTML/CSS/JavaScript
Abra no navegador:

- `http://localhost/Jovem-Aprendiz-Senac/<pasta>/<arquivo>.html`

### 3) Rodar exemplos PHP
1. Garanta que o **Apache** do XAMPP está ligado.
2. Acesse no navegador:

- `http://localhost/Jovem-Aprendiz-Senac/<pasta>/<arquivo>.php`

> Observação: para `GET/POST` (formulários e requisições), os links/ações (`action`) precisam apontar para os arquivos `.php` corretos.

### 4) (Opcional) Banco de Dados (MySQL)
A pasta **`Banco de Dados/`** contém arquivos `.sql` e materiais de diagramas.

Fluxo típico:
1. Iniciar o **MySQL** no XAMPP
2. Criar um banco (conforme o script)
3. Importar os arquivos `.sql` no phpMyAdmin

## Estrutura do repositório

- **`AtividadesPHP/`**: exercícios e exemplos em **PHP** (inclui `GET/POST`).
- **`Banco de Dados/`**: scripts SQL e materiais de modelagem/diagramas.
- **`Bootstrap/`**: páginas e atividades usando **Bootstrap**.
- **`Formulários/`**: exemplos de formulários.
- **`JavaScript/`**: atividades e exemplos em JavaScript.
- **`Exemplo de página/`**: páginas estáticas demonstrativas (HTML/CSS).

---

## Como navegar (dica rápida)

1. Comece por **HTML/CSS/JavaScript** para entender a base.
2. Depois, verifique **`AtividadesPHP/`** para ver lógica e manipulação com backend.
3. Por fim, quando houver persistência, use **`Banco de Dados/`** para scripts e modelos.
